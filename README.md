# asmr
A x86_64 macho64 assembly implementation of a subset of the libc library.
functions added:
- abs
- atoi
- bzero
- cat
- fact
- fib
- isalnum
- isalpha
- isascii
- isdigit
- isprint
- memcpy
- memset
- puts
- strcat
- strdup
- stritter
- strlen
- tolower
- toupper

## to build
`make`

## to test
`make test`
test compiles libftasm with main.c
